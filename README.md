
# Autopay SDK

Paying in a mobile app? We make it simple.  
**Autopay SDK** is a ready-to-use payment solution that can be easily integrated into Android and iOS applications. The SDK supports multiple languages (including English and German) and offers flexible UI customization, allowing it to match the visual style of your app.

Curious how it works before integration? Explore the Autopay SDK Demo to see different implementation variants in action.
